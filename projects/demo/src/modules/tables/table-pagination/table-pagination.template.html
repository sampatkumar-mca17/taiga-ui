Component to show pagination in table footer

You can customize the component via DI-token TUI_TABLE_PAGINATION_OPTIONS .

Use function tuiTablePaginationOptionsProvider to provide new value of this token.

Total amount of items/lines in the table. Items/Lines per page.

Uses the first element inside items by default.

Use paginationChange instead Current page.

Indexing starts at zero.

Use paginationChange instead Options to select amount of lines per page. Emits the selected page when user navigates by spin buttons or selects new size of items per page. Use paginationChange instead Emits the new size -property (user selects new amount of lines per page). Use paginationChange instead Event emits when page or size changes.

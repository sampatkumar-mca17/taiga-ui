Sanitizer

If you use unsafe icons or Editor component you need to provide TUI_SANITIZER token, which is responsible for removing malicious code from the svg. We recommend to use NgDompurify as a sanitizer. This library implements DOMPurify as Angular Sanitizer or Pipe. It delegates sanitizing to DOMPurify and supports the same configuration.

Don't forget provide TUI_SANITIZER if you want to use Svg -component with:

  • icon which was imported from source code
  • custom icons that you don't trust

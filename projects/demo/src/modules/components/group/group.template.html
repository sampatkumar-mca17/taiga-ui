A directive for grouping other components. For example, textfields and buttons .
Items take all available width. If you want to prevent it, use class=" tui-group__auto-width-item " .

If you want item to inherit border radius, use class class=" tui-group__inherit-item "

Enable adaptive mode (for mobile) Remove margin between items ( 2px by default) The first and the last items are rounded Horizontal or vertical direction of group Size of rounding
